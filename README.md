# spam-sms-classifier
This project is a simple SMS Spam Detection System built using Python, Scikit-Learn, and TF-IDF text vectorization. It classifies SMS messages as Spam or Ham (Not Spam) using a Naive Bayes classifier, which is one of the most effective algorithms for text classification.
